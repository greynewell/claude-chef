Install Claude Chef

Add Claude Chef to your home screen for instant access, or install the CLI plugin for recipe creation from your terminal.

Add to Home Screen

Claude Chef works as a progressive web app. Install it for a native app-like experience with offline support.

iPhone / iPad (Safari)

  1. Open claudechef.com in Safari
  2. Tap the Share button (square with arrow) at the bottom of the screen
  3. Scroll down and tap Add to Home Screen
  4. Tap Add in the top right corner

Android (Chrome)

  1. Open claudechef.com in Chrome
  2. Tap the three-dot menu in the top right
  3. Tap Add to Home screen or Install app
  4. Tap Add to confirm

Desktop (Chrome / Edge)

  1. Open claudechef.com in Chrome or Edge
  2. Click the install icon in the address bar (monitor with down arrow), or open the browser menu
  3. Click Install Claude Chef
  4. The app will open in its own window and appear in your app launcher

Claude Code CLI Plugin

If you use Claude Code, you can install Claude Chef as a plugin to create, enrich, and submit recipes directly from the terminal.

Install the Plugin

claude install-plugin github:greynewell/claude-chef

Create a Recipe

/chef create "Spicy Garlic Noodles"

Creates a new recipe markdown file with frontmatter template in the recipes directory.

Enrich a Recipe

/chef enrich spicy-garlic-noodles

Uses AI to add shopping terms, gear recommendations, cooking tips, and a coaching prompt.

Submit for Review

/chef submit

Lints the recipe and prepares it for submission to the repository.
