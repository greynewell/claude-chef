How Claude Chef Works

Claude Chef combines thousands of community-contributed recipes with AI-powered tools to make cooking more accessible, more fun, and more delicious.

Features

Cook with AI

Every recipe includes a one-click "Cook with AI" prompt you can paste into Claude, ChatGPT, or any AI assistant. Get step-by-step guidance, substitution ideas, and answers to cooking questions in real time.

Smart Ingredient Shopping

Each recipe page includes normalized shopping links for every ingredient. Quickly find what you need on Amazon, Walmart, or your preferred store. Adjust servings with the slider and quantities update automatically.

Gear Recommendations

Recipes include recommended tools and equipment with shopping links. Whether you need a cast iron skillet or a stand mixer, Claude Chef helps you find the right gear.

Recipe Pairings

Discover what goes together. Each recipe suggests complementary dishes so you can build complete meals. Pair your main course with the perfect side, salad, or dessert.

Browse by Category, Cuisine & More

Explore recipes organized across multiple dimensions: category, cuisine, ingredients, flavors, sauces, tools, skill level, and contributor. Find exactly what you are looking for or discover something new.

Claude Code CLI Plugin

Claude Chef is also a Claude Code plugin. Install it to create, enrich, and submit recipes directly from your terminal.

claude install-plugin github:greynewell/claude-chef

Then use it with:

/chef create "Spicy Garlic Noodles"
/chef enrich spicy-garlic-noodles
/chef submit

Open Source

Claude Chef is fully open source under the CC0-1.0 license. The entire site is static HTML with zero runtime dependencies — no framework, no build step for the output, no client-side libraries.
